 # LOGIN SIMPLE LARAVEL

Sistema de autenticación básico desarrollado con Laravel.
Incluye registro, inicio de sesión y protección de rutas.

---

 ### Enfoque

Base para sistemas reales como:

- Paneles administrativos
- Ecommerce
- Aplicaciones con usuarios autenticados

---

 ### Qué demuestra

- Flujo completo de autenticación
- Manejo de sesiones
- Protección de rutas
- Integración con base de datos

---

 ### Tecnologías

- Laravel
- PHP
- SQLite

---

  ### Requisitos

- Git
- PHP
- Composer

---

  ### Instalación

git clone https://github.com/FedericoZorrillaOficial/login_simple

cd login_simple

composer install

cp .env.example .env

---

 ### Ejecución

php artisan key:generate
php artisan migrate --force
php artisan serve

Abrir en navegador:

http://127.0.0.1:8000
